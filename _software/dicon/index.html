DiCon: Disease Control System

The Disease Control System DiCon (pronounced [ˈdaɪkɒn]) provides a general optimization framework, with a special focus on distributed, parallel epidemic disease simulation with policy optimization.

Documentation

Full user documentation is in the making. For now, the following documents are provided, in addition to the README and INSTALL files which ship with DiCon core.

Name File Released Description
Introduction DiCon Introduction.pdf 2009-11-24 Introductory slides on DiCon. General overview of the system, design goals, and sample implementations of a very simple (non-disease) simulator.
Master's Thesis Sebastian Goll … 2009.pdf 2009-12-04 Thesis describing the inner workings of the implementation of DiCon. Particularly useful are Appendix A (reference on the DiCon configuration) and Section 3.4.1 (description of argument and schedule specifiers used in configuration).
